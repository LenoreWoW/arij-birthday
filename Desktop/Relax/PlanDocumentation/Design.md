Brand Voice & Tone

Friendly expert, minimal jargon

Action-oriented CTAs (“Book now,” “Add favorite”)

Color, Typography, Spacing

Palette: White #FFF, Teal #D7F4E2, Blue #E3F2FD

Type: Inter; body 16px, headings 20px

Spacing: 16px gutters, 24px card padding

Layout Best Practices

Mobile-first grid; 2-column web dashboard

Cards with 2xl corner radius, soft shadows

Stepper UI for multi-step flows

Accessibility Must-Dos

AA contrast ≥4.5:1, semantic HTML, ARIA labels, visible focus states

Content Style Guide

Headings ≤5 words, Title Case

Bullets as fragments, no periods

Descriptive links (“View promotions”)
